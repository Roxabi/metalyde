Music & Sound Design

Minimal and precise — the sound of a control room, not a hype reel. A low steady pulse under the problem; a single clean tick as each margin row lands; one confident resolve on the tagline. No epic swell. The brand voice is calm under load — the audio must project control, not urgency. Reference: ambient-techno restraint (e.g. Jon Hopkins, low-mix).

Notes for Social Format

Square 1:1 (X · LinkedIn feed)

  • 1080 × 1080 — safe zone 900 centered
  • SyncedCaptions mandatory (muted autoplay)
  • Signal-red brand band top or bottom, 40px
  • System sans caption, min 30px for legibility
  • Export two versions — with / without burned captions

Vertical 9:16 (Reels / Stories / Shorts)

  • 1080 × 1920 — content recentered for vertical
  • Panel / sheet takes ~60% height, centered
  • VO text as large captions, top 20%
  • METALYDE wordmark bottom-right, always visible
  • Render as separate composition from 1:1

Production Workflow
Pipeline
End-to-end, from VO script to platform-ready export. Engine: roxabi-production (frame-based, useCurrentFrame) + voiceCLI.
1
Write VO scripts
Finalize narration line-by-line from this brief. Short sentences — operator register, no hype (the brand voice is calm under load). Target ~120 wpm. One segment per scene, 500 ms gap between segments; mark emotion hints inline as HTML comments.
Output: vo-launch.md · vo-demo.md · vo-clips.md
2
Generate voice — voiceCLI (TTS)
Engine qwen-fast (validated on M₂ RTX 5070Ti). Voice profile: calm, precise, low-affect — operator-to-operator, never salesy. Slight pause on the numbers ($4,200 · 57% · 500 hrs). Crossfade 80 ms. Output WAV for render, OGG/Opus for preview.
voicecli tts --engine qwen-fast --voice <operator> --input vo-launch.md --out vo-launch.wav
3
Transcribe for word-level timestamps
faster-whisper large-v3-turbo → word-level timestamp JSON consumed by SyncedCaptions for frame-accurate captions. Also validates VO timing against each scene's frame budget — trim the script if any segment runs long.
voicecli stt --model large-v3-turbo --word-timestamps --input vo-launch.wav --out vo-launch.ts.json
4
Build roxabi-production compositions
Three compositions: MetalydeLaunchTrailer, MetalydeDemo, MetalydeSocialClips (3 sub-comps: 1:1 + 9:16). Register in Root. Palette constants: PAPER="#f5f5f0" INK="#1a1a1a" SIGNAL="#e63000" BORDER="#d8d8d0". Heading constant HEADING_FONT = system sans stack. All motion via useCurrentFrame() — zero CSS animation.
src/showcase/MetalydeLaunchTrailer.tsx · MetalydeDemo.tsx · MetalydeSocialClips.tsx
5
kit-metalyde custom components
Swiss-control-room variants in src/kit-metalyde/ — ControlRoomFrame (paper sheet + hairline grid + top status bar), MarginPanel (the row-reveal client-margin table), SeamFlow (four-cards → one-sheet collapse), StatBlock (big signal-red figure + mono label), LogoGridBuild (M-grid mark assembly). These wrap base kits — styled compositions, no new animation primitives.
src/kit-metalyde/ControlRoomFrame.tsx · MarginPanel.tsx · SeamFlow.tsx · StatBlock.tsx
6
Render — roxabi-production CLI
Start the dev server on port 3001 (renderer hardcodes it). Render one composition at a time (saturates the machine); concurrency = CPU cores. Typecheck first to catch component errors before a long render. CRF 18 master, CRF 23 distribution copies. Verify frame count against the timing spec before committing.
npm run render -- MetalydeLaunchTrailer out/metalyde-launch-v0.1.mp4
7
Post-production & distribution
Thumbnail: export the S01 hook frame ($0 profit) for landing/X/LinkedIn cards. Burn SyncedCaptions into the social clips. X/LinkedIn: encode ≤5 Mbps. Export a 1:1 square for feed autoplay and a 9:16 for Reels/Shorts. Verify all color profiles are sRGB before upload.
ffmpeg -i out/metalyde-launch-v0.1.mp4 -b:v 4M -maxrate 5M out/metalyde-launch-x.mp4

Brand fidelity — lock before render
  • Fonts: system sans stack only (no web fonts). Centralize as a HEADING_FONT constant — font drift is the #1 video brand risk.
  • Palette: every hex cross-checked to Direction A — PAPER #f5f5f0 · INK #1a1a1a · SIGNAL #e63000 · BORDER #d8d8d0. No glow, no gradients.
  • No Lyra carry-over: the forge orange #e85d04 and dark obsidian are a different brand — never appear here. Metalyde signal is #e63000 on paper.
  • Logo: the M-grid mark must match the wordmark SVG exactly (module geometry), not a reinterpretation.
  • Closing card: centered logo + wordmark + “The OS for a marketing agency.” + sub + CTA — never a small watermark.
  • Numbers are sourced: 57% / $1–5K (Ignition 2025) · ~60% utilization (SoDA 2024) · $4,200 scope/week — keep them accurate on screen.
